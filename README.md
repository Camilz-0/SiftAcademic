# SiftAcademic
Un agente inteligente para el navegador diseñado para optimizar la investigación académica mediante análisis semántico en tiempo real, detección de contradicciones y automatización de citas bibliográficas.

## Auditoría de Logs
He implementado un sistema de registro de eventos (logging) para rastrear automáticamente la carga de documentos y el análisis de compatibilidad de SiftAcademic, permitiendo depurar errores de procesamiento y monitorear el flujo de datos de manera eficiente.
